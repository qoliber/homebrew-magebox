# Homebrew Tap for MageBox

This is the official Homebrew tap for [MageBox](https://magebox.dev) - a fast, native Magento development environment.

## Installation

```bash
brew tap qoliber/magebox
brew install magebox
```

Or install directly:

```bash
brew install qoliber/magebox/magebox
```

## Updating

```bash
brew update
brew upgrade magebox
```
